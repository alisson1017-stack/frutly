<h2>O que é?</h2>
<p>Um projeto simples para praticar lógica de programação, pensamento computacional e o uso de dicionários em Python.</p>
<p>Um programa feito para o cadastro de frutas e seus dados —nome, valor, id e margem de lucro— e as quantidades de frutas vendidas por mês.</p>

<br>
<p>Nenhum workflow foi utilizado, pois, até o momento em que eu estou escrevendo esse README, nem entendi direito o que é um workflow.</p>

<br>

<h2>Uso de Inteligência Artificial</h2>

![python](https://img.shields.io/badge/AI-LOW-orange?style=for-the-badge)

<p>Como o foco do projeto é o aprendizado, eu limitei o uso de IA a eventuais consultas e debbugings, quando eu desistia de tentar achar sozinho.</p>
<p>No início do desenvolvimento do projeto o uso de IA era feito em quase toda etapa.<br> A partir da metade do projeto, a necessidade pelo uso de IA foi diminuída a praticamente zero.</p>

<br>

<h2>Funcionalidades</h2>

<p>O programa permite as seguintes ações:
<ul>
<li>Adicionar novas frutas</li>
<li>Adicionar novos meses</li>
<li>Listar todas as frutas</li>
<li>Listar todos os meses</li>
<li>Remover Fruta</li>
<li>Remover Mês</li>
<li>Comparar:</li>

<ul>
<li>Frutas</li>

<ul>
<li>por quantidade vendida</li>
<li>por valor total</li>
<li>por valor final (margem de lucro)</li>
</ul>

<li>Meses</li>

<ul>
<li>por quantidade vendida</li>
<li>por valor total</li>
<li>por valor final (margem de lucro)</li>
</ul>
</ul>
</ul>
</p>
